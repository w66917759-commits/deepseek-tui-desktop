# Superpowers

This is the complete Superpowers skill pack. Treat this directory as one bundled
skill: enabling `superpowers` enables the whole pack together.

## Pack Contents

- `using-superpowers`
- `brainstorming`
- `test-driven-development`
- `systematic-debugging`
- `writing-plans`
- `executing-plans`
- `subagent-driven-development`
- `dispatching-parallel-agents`
- `requesting-code-review`
- `receiving-code-review`
- `verification-before-completion`
- `finishing-a-development-branch`
- `using-git-worktrees`
- `writing-skills`

## Usage

Start with `using-superpowers/SKILL.md`. When that workflow calls for another
Superpowers skill, read the matching bundled subdirectory inside this same pack.

Do not treat the subdirectories as separately enabled product toggles. The app
turns this pack on or off as a single `superpowers` skill.
